# RepCounters
A research and basic implement for Rep counters of Push-ups 
and Pull-ups. 

# Tech Stack
Apple/Android Users can use this RepCounters app to track their 
push-ups, and pull-ups very easily. 

- Frontend: Typescript, react, and swift/Android.
- Backend: Go.
